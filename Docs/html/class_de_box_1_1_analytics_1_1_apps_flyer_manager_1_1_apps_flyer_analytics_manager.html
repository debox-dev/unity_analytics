Detailed Description

An IAnalyticsManager that uses the AppsFlyer SDK as the backend for tracking events

To use this manager you must install the AppsFlyer SDK. This manager should work with the V5 SDK.

AppsFlyer do not supply their SDK via the new Unity Packaging system yet, Because of that, depending on what kind of project you are working on (UPackage or Game) you may have to create an assembly definition for the AppsFlyer SDK. If so, call it "AppsFlyerSDK".

Make sure to add DEBOX_ANALYTICS_APPSFLYER to your symbol definitions when using this manager
